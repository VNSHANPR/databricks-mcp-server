Available Tools

get_inflation

Latest CPI inflation rate for any country — World Bank API.

get_macro_indicators

CPI, GDP growth, real interest rate, unemployment — World Bank API.

analyze_economic_webpage

Fetch a public webpage and analyse it using Claude_for_Analysis on Databricks Model Serving.

list_abs_dataflows

List available Australian Bureau of Statistics (ABS) SDMX dataflows with key syntax hints.

get_abs_data

Fetch time-series data from the ABS SDMX REST API — CPI, Labour Force, Wage Price Index, GDP, Property Prices, Trade, and more. No API key required.
